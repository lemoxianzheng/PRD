# 辅助线
> 一个页面中最多添加20个辅助线
![](media/15365670210813/15365691557912.jpg)
![](media/15365670210813/15365691752142.jpg)

可以设置辅助线的线型、线条的粗细、线条的颜色

线型可选3种：直线（——————————）、点线（········）、虚线（---------）三种
线条粗细：1rpx-10rpx（10个整数值）

线条的颜色，可选的值如下表
